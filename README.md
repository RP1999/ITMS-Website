# ITMS - Intelligent Traffic Management System

![ITMS Banner](https://RP1999.github.io/ITMS-Website/src/assets/logo-icon.png)

A modern, high-performance web application for the **Intelligent Traffic Management System**, designed with a premium dark-mode aesthetic, smooth animations, and interactive components.

## 🚀 Features
- **Anti-Gravity Background:** Dynamic particle system that reacts to user interaction.
- **Modern UI:** Built with React, Framer Motion, and GSAP for a state-of-the-art user experience.
- **Interactive Milestones:** Track project progress through a glassmorphism-styled timeline.
- **Document Management:** Direct access to Project Charters, Proposals, and Final Reports.
- **Team Showcase:** Professional profiles for supervisors and research team members.

## 🛠️ Technology Stack
- **Frontend:** React 19, Vite
- **Animations:** Framer Motion, GSAP
- **Icons:** Lucide-React
- **Deployment:** GitHub Pages (via GitHub Actions)

## 📁 Project Structure
- `src/assets/`: Logos and team member images.
- `public/docs/`: Project documentation and presentations.
- `.github/workflows/`: Automatic deployment configuration.
